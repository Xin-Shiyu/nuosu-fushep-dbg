# Nuosu FUSHEP
ꃚꎹ / FUSHEP（發音 /fv̄.ʂɯ̂/）——涼山規範彝文檢字編輯器

# 简介
**涼山規範彝文**是用於書寫**彝語北部方言**（諾蘇話）的官方文字，亦是目前唯一一種獲得國家認可的彝語文字。

該彝文系統是由傳統的意音結合型彝文修訂而来，每種音節只取一个字元，一字對應一音節，包括756個用於表示固有詞語音節的字元、63個用於表示漢語借詞音節的字元、345個帶變調符號的字元（在Unicode中分別編碼），以及一個音節重複符號（彝文方案中稱為“替音符號”）。
去除帶變調符號的字元以及音節重複符號後，總計有819個字元，因此民間亦称此方案文字為“819彝文”。

此種彝文在四川涼山彝族自治州具有官方地位并廣泛使用，在雲南的紅河哈尼族彝族自治州亦可見若干用例（儘管当地的彝語并非北部方言）。
其他彝語使用地区常使用各种傳統的意音型彝文以及基於此修訂規範而成的現代形式，例如“雲南規範彝文”等，此類彝文尚未得到Unicode編碼。

同為彝語支的哈尼語、傈僳語等語言，其使用族群在民族認同上有別於彝族，傳統上通常不使用彝文書寫。
