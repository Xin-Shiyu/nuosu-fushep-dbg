# Nuosu FUSHEP
ꃚꎹ / FUSHEP（/fv̄.ʂɯ̂/ フシュ）——涼山規範彝（イ）文字検索・編集ツール

# 概要
**涼山規範彝文字**は、**イ語北部方言**（ノス語）を表記するために用いられる公式文字であり、現在において国家から正式に承認された唯一のイ語文字です。

この体系は、従来の（漢字のような）意音型の文字を改訂したもので、各音節につき1文字のみを採用し、1字が1音節に対応します。固有語の音節を表す756字、中国語借用語の音節を表す63字、声調変化記号付きの文字345字、および繰り返し記号を含みます。
声調記号付き文字と繰り返し記号を除くと、合計819字となるため、民間では「819彝文」とも呼んでいます。

この彝文は四川省涼山イ族自治州において公的な地位を持ち、広く使用されています。また、雲南省紅河ハニ族イ族自治州でも若干の使用例が見られます（ただし、当地のイ語は北部方言ではありません）。
他のイ語使用地域では、各種の伝統的な意音型彝文およびそれを基に標準化された体系（例えば「雲南規範彝文」など）が使用されていますが、これらの種類の彝文は未だUnicodeの標準規格に符号化されていません。

同じイ語派に属するハニ語、リス語などの言語を話す集団は、民族的アイデンティティにおいてイ族とは区別され、伝統的に彝文を使用しないのが一般的です。
