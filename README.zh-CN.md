# Nuosu FUSHEP
ꃚꎹ / FUSHEP（发音 /fv̄.ʂɯ̂/）——凉山规范彝文检字编辑器

# 简介
**凉山规范彝文**是用于书写**彝语北部方言**（诺苏话）的官方文字，也是目前唯一一种获得国家认可的彝语文字。

该彝文系统是由传统的意音结合型彝文修订而来，每种音节只取一个字符，一字对应一音节，包括了756个用于表示固有词语音节的字符、63个用于表示汉语借词音节的字符、345个带变调符号的字符（在Unicode中分开编码），以及一个音节重复符号（方案称为“替音符号”）。
去除带变调符号的字符以及音节重复符号后，一共有819个字符，因此民间亦称此方案的文字为“819彝文”。

此种彝文在四川凉山彝族自治州具有官方地位并广泛使用，在云南的红河哈尼族彝族自治州亦可见若干用例（尽管当地的彝语并非北部方言）。
其他彝语使用地区通常使用各种传统的意音型彝文以及基于此的现代规范形式，例如“云南规范彝文”等，这些种类的彝文并未得到Unicode编码。

同为彝语支的哈尼语、傈僳语等语言，其使用族群在民族认同上有别于彝族，传统上通常不使用彝文书写。
